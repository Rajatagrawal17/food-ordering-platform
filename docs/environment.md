# Environment Setup

## Root

- `NODE_ENV`
- `PORT`
- `MONGODB_URI`
- `CLIENT_ORIGIN`

## Auth

- `JWT_ACCESS_SECRET`
- `JWT_REFRESH_SECRET`
- `JWT_ACCESS_EXPIRES_IN`
- `JWT_REFRESH_EXPIRES_IN`

## Payments

- `RAZORPAY_KEY_ID`
- `RAZORPAY_KEY_SECRET`

## Media

- `CLOUDINARY_CLOUD_NAME`
- `CLOUDINARY_API_KEY`
- `CLOUDINARY_API_SECRET`

## Security

- `CSRF_SECRET`
- `LOG_LEVEL`
- `SENTRY_DSN`

Copy `.env.example` to `.env` before starting the app.
