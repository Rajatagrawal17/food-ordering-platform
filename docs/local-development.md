# Local Development

1. Install dependencies with `npm install`.
2. Create `.env` from `.env.example`.
3. Start MongoDB locally or via Docker.
4. Seed the database with `npm run seed --workspace server`.
5. Start both apps with `npm run dev`.
6. Open the client at `http://localhost:5173`.

## Useful Commands

- `npm run lint`
- `npm run test:coverage`
- `npm run build`
